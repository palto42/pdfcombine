# Getting pdfcombine

## Using conda

```bash
conda install -c conda-forge pdfcombine
```

## From source

```bash
# Download pdfcombine
git checkout https://github.com/tdegeus/pdfcombine.git
cd pdfcombine

# Install
cmake .
make install
```
